Assignment for Lecture 3

 

 

  1. Write a public class called “ServerApp” and do the following:
    1. Place the class in this package name: javatraining.example.multithreads.chat
      NOTE: make sure your package name looks EXACTLY as you see it here, Java is case- sensitive
    2. Declare an instance variable called isListening of primitive datatype boolean with an access modifier of private.
    3. Create a main method.
    4. Inside the main method, create a new instance of the ServerApp class.
    5. Inside the main method, assign isListening to have the value of "true".
      NOTE: in order to access the isListening variable inside a static method (main), you will need to use the object of the ServerApp class that you created in Step d. You must do this because isListening is an instance variable and NOT a class (static) variable.
    6. Inside the main method, create a "while" loop whose condition is the evaluation of the value of isListening. Don't forget to use the ServerApp object to access the isListening variable!!
    7. Inside the while loop, print this statement to the console, "Inside the while loop as long as isListening is true!"
    8. After the statement and still in the while loop, set isListening to "false". Don't forget to use the ServerApp object to access the isListening variable!!
    9. Make (compile) your source code (.java file)
    10. Run your ServerApp.java file and see the resulting output.
    11. Email the ServerApp.java file to the teacher. DO NOT email the .class file or the .jpx file.
