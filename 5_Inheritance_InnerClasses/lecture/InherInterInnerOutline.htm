II.                 Polymorphism

a.       Message is the same but the object behaves differently – general definition

b.      Two types of polymorphism:  Polymorphic Inheritance (true) and Ad-hoc

                                                                           i.      Sub-type polymorphism (true polymorphism)

 

                                                                         ii.      Ad-hoc polymorphism – this occurs when methods are overloaded within a class; this means that methods in a class have the same name but different signatures.
